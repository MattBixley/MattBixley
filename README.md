# Matt Bixley

- **eResearch Infrastructure** — building and supporting research computing platforms (NeSI, OOD, RStudio Server)
- **Bioinformatics** — genomics workflows, data wrangling, next-gen sequencing analysis
- **Data Science** — R, Python, visualisation, reproducible research
- **Teaching** — [Carpentries](https://carpentries.org) instructor; workshops in R, genomics, and data analysis
